## ★ About Me

I'm a third-year **Computer Science & Mathematics** student at the **University of Victoria**, interested in AI, data, and project management.

Some of what I've been up to:

- 🔧 **FortisBC** — Building real-time control software for industrial systems, including safety logic, fault detection, and sensor integration
- 🏥 **Fraser Health Authority** — Prototyped a low-latency pediatric MRI simulator end-to-end, from backend APIs to the frontend, in a safety-critical environment
- 🔬 **Engineering Faculty Research Assistant** — Optimizing single-cell RNA-sequencing pipelines and building ML models for biological data, working toward a publication
- 🛠️ **UVic Mcpherson Library Software Assistant** — Supporting 3D modeling and prototyping workflows using Fusion 360, SolidWorks, and Blender

When I'm not deep in a project, I'm probably watching crime documentaries, at the gym, or getting lost in a good playlist. 🎵
